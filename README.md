# 🎁 Birthday Gift Generator

A personalized, fantasy-themed web experience created to celebrate special occasions. This project features interactive "gift boxes" that reveal fun reactions when hovered over, blending a clean UI with a magical aesthetic.

## ✨ Features

* **Interactive Reveals:** Uses CSS transitions and hover states to reveal hidden birthday reactions.
* **Fantasy Aesthetic:** Customized with a sophisticated color palette (`#F5E6BE`, `#722F37`, `#D4AF37`) and magical typography (Cinzel & Quicksand).
* **Responsive Design:** Optimized for different screen sizes to ensure the celebration looks great on any device.
* **Modern CSS:** Implements advanced techniques like `background-origin` and `clip-path` for gradient borders.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for better accessibility.
* **CSS3:** Custom properties, linear gradients, and interactive animations.
* **Google Fonts:** Utilizing *Cinzel* for a bold, magical feel and *Quicksand* for readability.

## 🚀 How to Use

1.  **Clone the repo:**
    ```bash
    git clone [https://github.com/BashJuno/birthday-gift-generator.git](https://github.com/BashJuno/birthday-gift-generator.git)
    ```

2.  **Customize the content:**
    Update the `index.html` with the name, age, and date of the birthday person.

3.  **Add your reactions:**
    Place your images or GIFs in the `./images/` folder and link them in your `styles.css` file.

4.  **Open in Browser:**
    Launch `index.html` and enjoy the celebration!
